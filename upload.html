Upload Instructions:
  1. Select or drag & drop PDF files
  2. Fill in publication details for each file
  3. Click "Upload All Publications"
  4. Files will be automatically uploaded to the pdfs/ folder
  5. You'll be redirected to the publications page to see your new uploads

Note: Make sure both servers are running:

  • Main site: http://localhost:8000
  • Upload server: http://localhost:8001
